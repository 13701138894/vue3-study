## Start
 - NodeJS 10.0 need
 https://nodejs.org/en/
 - Clone or download this repository
 - Enter your local directory, and install dependencies:

``` bash
npm install -g live-server
npm start
```
如果不能自动打开浏览器 请浏览器访问: http://localhost:8081
